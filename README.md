## usage
You can specify how many bottles of water there are on the wall by using an integer greater than 0 as argv[1].<br/>
eg:
```
./water 100     # for 100 bottles of water on the wall (default value)
./water 10000   # as ParrotX2 once said, "No, it's Thursday bro"
```
